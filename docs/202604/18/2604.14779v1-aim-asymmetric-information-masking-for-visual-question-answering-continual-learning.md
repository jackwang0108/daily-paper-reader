## 摘要
在持续视觉问答（VQA）中，现有的持续学习（CL）方法大多是为对称的单模态架构设计的。然而，现代视觉语言模型（VLM）违背了这一假设，因为其可训练组件本质上是非对称的。这种结构不匹配使得 VLM 在从连续数据流中学习时极易发生灾难性遗忘。具体而言，这种非对称性导致标准的全局正则化在优化过程中偏向庞大的语言解码器，使得规模较小但至关重要的视觉投影层极易受到干扰。因此，这种局部退化导致了组合推理能力的严重丧失。为了解决这一问题，我们提出了非对称信息掩码（AIM），通过根据特定模态的敏感性应用针对性掩码，来平衡稳定性和塑性。在持续 VQA 设置下对 VQA v2 和 GQA 进行的实验表明，AIM 在平均性能（AP）和平均遗忘率（AF）方面均达到了最先进水平，同时能更好地保持对新颖技能-概念组合的泛化能力。

## Abstract
In continual visual question answering (VQA), existing Continual Learning (CL) methods are mostly built for symmetric, unimodal architectures. However, modern Vision-Language Models (VLMs) violate this assumption, as their trainable components are inherently asymmetric. This structural mismatch renders VLMs highly prone to catastrophic forgetting when learning from continuous data streams. Specifically, the asymmetry causes standard global regularization to favor the massive language decoder during optimization, leaving the smaller but critical visual projection layers highly vulnerable to interference. Consequently, this localized degradation leads to a severe loss of compositional reasoning capabilities. To address this, we propose Asymmetric Information Masking (AIM), which balances stability and plasticity by applying targeted masks based on modality-specific sensitivity. Experiments on VQA v2 and GQA under continual VQA settings show that AIM achieves state-of-the-art performance in both Average Performance (AP) and Average Forgetting (AF), while better preserving generalization to novel skill-concept compositions.

*   **研究背景**：视觉语言模型（VLMs）在处理连续数据流时面临严重的“灾难性遗忘”问题。
*   **核心挑战（非对称困境）**：现代 VLM 的结构本质上是非对称的。通常视觉骨干网络被冻结，只有轻量级的**视觉投影层（Projector）**和庞大的**语言解码器（LLM）**参与训练。
*   **发现**：传统的持续学习正则化方法（如 EWC）假设参数空间是均匀对称的。这导致优化过程中梯度被参数量巨大的语言端主导，而脆弱且关键的视觉投影层（信息瓶颈）因缺乏足够保护而发生退化，最终导致模型丧失“组合推理”能力（即无法将学到的技能应用到新概念上）。

### 2. 方法论：非对称信息掩码（AIM）
AIM 的核心思想是根据不同模态子空间的敏感度和容量，实施差异化的参数保护策略。
*   **架构分区**：将可训练参数分为三个子空间：视觉投影子空间（$\Theta_{vis}$）、跨模态对齐子空间（$\Theta_{shared}$）和认知推理子空间（$\Theta_{text}$）。
*   **参数重要性估计**：利用经验费舍尔信息矩阵（FIM）衡量参数敏感度。为了防止参数空间被过度约束，提出了一种**递归最大值聚合函数**（Max Aggregation），而非传统的累加方式。
*   **生成非对称掩码**：为不同子空间设置不同的保护比例 $\rho_m$。
    *   通过计算各子空间内 Fisher 值的分位数来确定动态阈值。
    *   生成二进制梯度掩码：敏感度高的关键参数被冻结（保持稳定性），敏感度低的参数允许更新（保持塑性）。
*   **优化流程**：在反向传播时，将掩码直接作用于原始梯度，并结合轻量级的经验回放（Memory Replay）来巩固历史知识。

### 3. 实验设计
*   **数据集与场景**：
    *   **VQA v2 (语言增量)**：按问题类型分为10个任务，重点测试“组合泛化”（COMP），即在未见过的技能-概念组合上的表现。
    *   **GQA (场景增量)**：按环境场景（如户外、交通等）分为6个任务，测试模型在视觉域偏移下的稳定性。
*   **基准方法（Baselines）**：
    *   正则化类：EWC, LwF, MAS。
    *   回放类：ER, DER。
    *   多模态专用类：VQACL, QUAD。
*   **评估指标**：平均性能（AP）和平均遗忘率（AF）。

### 4. 资源与算力
*   **硬件**：使用单张 **NVIDIA A800 GPU**。
*   **模型架构**：以预训练的 **VL-T5** 为基础模型，并在扩展实验中使用了 **LLaVA-7B**（结合 LoRA 技术）。
*   **训练细节**：每个任务训练 3 个 epoch，Batch Size 为 80，使用 AdamW 优化器。
*   **开销**：AIM 引入的额外计算量极小。以 `q_judge` 任务为例，计算 Fisher 信息仅增加约 0.16% 的优化量，训练时间增加约 2.3%，显存占用增加约 3.8%（约 428MB）。

### 5. 实验数量与充分性
*   **实验规模**：
    *   在两个主流数据集上进行了完整的持续学习序列测试。
    *   针对 VQA v2 进行了 5 折交叉验证以评估组合泛化能力。
    *   进行了详尽的消融实验（掩码策略对比、聚合方式对比、有无内存对比）。
    *   进行了参数敏感性分析（不同 $\rho$ 值的组合）和任务顺序鲁棒性测试。
*   **评价**：实验设计非常充分且客观。作者不仅对比了传统 CL 方法，还对比了最新的多模态专用方法，并提供了 Joint Training（联合训练）作为上限参考。通过可视化任务评估矩阵和参数漂移轨迹，深入揭示了方法生效的内在逻辑。

### 6. 主要结论与发现
*   **非对称保护至关重要**：对语言端实施严格约束（高 $\rho_{text}$）同时保持视觉端适度塑性（低 $\rho_{vis}$）是平衡 VLM 稳定性和塑性的关键。
*   **性能领先**：AIM 在 VQA v2 和 GQA 上均取得了 SOTA 结果。在 VQA v2 标准测试中 AP 达到 43.35%，遗忘率仅为 1.56%，显著优于其他方法。
*   **组合泛化能力强**：AIM 在处理“从未见过的组合”时表现优异，证明了保护视觉投影层有助于维持解耦的特征表示，从而实现零样本组合推理。

### 7. 优点
*   **视角独特**：首次深入分析并量化了 VLM 在持续学习中的“双重非对称性”（容量不对称与脆弱性不对称）。
*   **即插即用**：不需要修改模型架构，不引入复杂的辅助网络或动态路由，计算开销极低。
*   **泛化性强**：实验证明该方法不仅适用于中等规模的 VL-T5，也能有效扩展到 7B 规模的大型多模态模型（LLaVA）。

### 8. 不足与局限
*   **超参数依赖**：虽然提供了推荐配置，但不同架构（如从 T5 切换到 LLaVA）可能仍需要手动调整各子空间的掩码比例 $\rho$。
*   **视觉骨干冻结假设**：研究基于视觉 Encoder 冻结的常见前提，若未来场景需要微调巨大的视觉骨干网络，该方法的子空间划分和掩码比例可能需要重新设计。
*   **任务边界依赖**：实验仍基于有明确任务边界的设置，对于完全无边界（Blurry Boundary）的持续学习场景探讨较少。
