## 摘要
基于提示的类增量学习方法通常构建一个由多个可训练键值提示组成的提示池，并执行实例级匹配以选择最合适的提示嵌入，这已展现出良好的应用前景。然而，现有方法面临若干局限性，包括固定的提示池、提示嵌入的人工选择以及在提示选择上对预训练骨干网络的强依赖。为了解决这些问题，我们提出了一种层重要性引导的双可扩展提示池（LDEPrompt），它支持自适应层选择以及提示池的动态冻结与扩展。在广泛使用的类增量学习基准上进行的广泛实验表明，LDEPrompt 达到了最先进的性能，验证了其有效性和可扩展性。

## Abstract
Prompt-based class-incremental learning methods typically construct a prompt pool consisting of multiple trainable key-prompts and perform instance-level matching to select the most suitable prompt embeddings, which has shown promising results. However, existing approaches face several limitations, including fixed prompt pools, manual selection of prompt embeddings, and strong reliance on the pretrained backbone for prompt selection. To address these issues, we propose a \textbf{L}ayer-importance guided \textbf{D}ual \textbf{E}xpandable \textbf{P}rompt Pool (\textbf{LDEPrompt}), which enables adaptive layer selection as well as dynamic freezing and expansion of the prompt pool. Extensive experiments on widely used class-incremental learning benchmarks demonstrate that LDEPrompt achieves state-of-the-art performance, validating its effectiveness and scalability.

### 1. 核心问题与研究动机
*   **核心问题**：如何在不存储旧数据（Exemplar-free）的情况下，利用预训练模型（PTM）高效地进行类增量学习（CIL），并克服现有 Prompt 方法的局限性。
*   **研究背景**：现有的基于 Prompt 的方法（如 L2P, DualPrompt）虽然有效，但存在三大痛点：
    1.  **提示池固定**：容量有限，难以应对不断增加的新任务。
    2.  **覆盖风险**：跨任务重复选择的 Prompt 容易被新知识覆盖，导致灾难性遗忘。
    3.  **骨干网络依赖**：Prompt 的选择过度依赖冻结的预训练模型特征，缺乏任务特定信息的引导。

### 2. 核心方法论
LDEPrompt 提出了**层重要性引导的双可扩展提示池**机制，核心包含以下三个部分：
*   **层重要性评估（Layer Importance Evaluation）**：
    *   在训练新任务前，通过预训练模型计算各层的**信息增益（Information Gain, IG）**。
    *   公式定义为：$IG(l) = I(x; h_l) - I(x; h_{l-1})$，即层输入与输出之间的互信息差。
    *   通过 Softmax 归一化得到重要性得分，仅在得分高于平均值的层插入 Prompt，实现自适应层选择。
*   **双可扩展提示池（Dual Expandable Prompt Pools）**：
    *   **全局池（Global Pool, $P_g$）**：存储所有已学任务的 Prompt，保持冻结以保留旧知识。
    *   **训练池（Training Pool, $P_c$）**：具有固定容量，用于当前任务的学习。
    *   **交互机制**：通过 [CLS] token 的相似度匹配，从全局池中检索最相关的 $s$ 个 Prompt 放入训练池（冻结状态），并随机初始化 $S-s$ 个新 Prompt（可训练状态）。
*   **动态扩展与优化**：
    *   任务训练结束后，新学习的 Prompt 会被合并进全局池，实现池容量的动态增长。
    *   使用标准的交叉熵损失函数对训练池中的 Prompt 和分类器进行优化。

### 6. 主要结论与发现

*   **知识保留**：双池结构有效地平衡了“新知识学习”与“旧知识保持”，显著缓解了灾难性遗忘。

### 7. 优点与亮点
*   **自适应性强**：不再手动指定 Prompt 插入位置，而是通过信息增益自动定位关键层。

*   **参数高效**：仅训练少量的 Prefix 向量，保持骨干网络冻结，符合大模型时代的微调范式。
