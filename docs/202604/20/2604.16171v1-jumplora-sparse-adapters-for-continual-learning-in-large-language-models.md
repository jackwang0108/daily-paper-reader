## 摘要
基于适配器（Adapter）的方法通过为每个任务顺序学习低秩更新矩阵，已成为大语言模型（LLMs）持续学习（CL）的一种高性价比方法。为了缓解灾难性遗忘，目前最先进的方法通过针对子空间或坐标维度的干扰，对新旧适配器之间的关系施加约束。在本文中，我们提出了 JumpLoRA，这是一个通过使用 JumpReLU 门控在低秩自适应（LoRA）模块中自适应诱导稀疏性的新颖框架。该方法实现了动态参数隔离，有助于防止任务干扰。我们证明了该方法具有高度的模块化，并与基于 LoRA 的持续学习方法兼容。具体而言，它显著提升了 IncLoRA 的性能，并超越了目前领先的持续学习方法 ELLA。

## Abstract
Adapter-based methods have become a cost-effective approach to continual learning (CL) for Large Language Models (LLMs), by sequentially learning a low-rank update matrix for each task. To mitigate catastrophic forgetting, state-of-the-art approaches impose constraints on new adapters with respect to the previous ones, by targeting either subspace or coordinate-wise interference. In this paper, we propose JumpLoRA, a novel framework to adaptively induce sparsity in the Low-Rank Adaptation (LoRA) blocks through the use of JumpReLU gating. The method achieves dynamic parameter isolation, which helps prevent task interference. We demonstrate that our method is highly modular and compatible with LoRA-based CL approaches. Specifically, it significantly boosts the performance of IncLoRA and outperforms the leading state-of-the-art CL method, ELLA.

论文聚焦于**大语言模型（LLM）在持续学习（Continual Learning, CL）中的灾难性遗忘问题**。
*   **背景**：LLM 通常是静态的，顺序学习新任务时会丢失旧知识。虽然 LoRA 等参数高效微调（PEFT）方法降低了计算成本，但简单的 LoRA 顺序训练仍会因任务间的参数干扰导致严重遗忘。
*   **核心问题**：现有的 CL 方法（如子空间投影或坐标约束）往往在密集参数空间操作，随着任务增加，约束空间变得极其拥挤，导致模型容量饱和或梯度干扰累积。
*   **动机**：通过引入**稀疏性（Sparsity）**来实现更彻底的参数隔离，让不同任务的适配器占据不重叠的参数坐标，从而在保持模型塑性的同时增强稳定性。

### 2. 方法论：核心思想与关键技术
JumpLoRA 的核心思想是利用 **JumpReLU 门控机制** 为 LoRA 适配器自适应地诱导稀疏性。

*   **核心思想**：在训练 LoRA 矩阵 $A$ 和 $B$ 的同时，学习一个阈值 $\tau$。只有当权重更新的幅度超过 $\tau$ 时，该更新才会被保留。这使得模型能自动识别并仅更新对当前任务最重要的参数。
*   **关键技术细节**：
    *   **JumpReLU 门控**：定义为 $JumpReLU_\tau(x) = x \cdot H(x - \tau)$，其中 $H$ 是赫维赛德阶跃函数。为了解决不可导问题，训练时使用直通估计器（STE）计算伪梯度。
    *   **渐进式调度（Scheduling）**：由于 LoRA 初始化时更新量为 0，直接应用阈值会阻断梯度。JumpLoRA 引入了凸插值机制，在训练初期进行全量更新，随后逐渐增加稀疏约束，直到训练结束时完全切换为稀疏更新。
    *   **阈值初始化**：在预设的迭代步数 $S_{start}$，根据权重幅度的分位数初始化 $\tau$，确保初始保留的参数量与 LoRA 原始参数量相当。
*   **算法流程**：为每个新任务注入新的 LoRA 层 $\rightarrow$ 密集训练 $\rightarrow$ 启动 JumpReLU 插值并学习阈值 $\rightarrow$ 最终获得稀疏适配器 $\rightarrow$ 将稀疏更新合并回基础权重。
