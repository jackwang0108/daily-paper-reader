## 摘要
大语言模型的有监督微调（SFT）通常面临任务干扰和灾难性遗忘的问题。近期的研究方法通过在训练过程中隔离任务关键参数来缓解这一问题。然而，这些方法是针对动态问题的静态解决方案，假设参数重要性一旦确定就保持不变。在这项工作中，我们通过实证研究证明，参数重要性在训练过程中表现出时间偏移。为此，我们提出了演进式参数隔离（Evolving Parameter Isolation, EPI），这是一种根据参数重要性的在线评估来调整隔离决策的微调框架。EPI 并非冻结固定的参数子集，而是利用基于梯度的信号定期更新隔离掩码，使模型能够保护新出现的任务关键参数，同时释放过时的参数以恢复可塑性。在多种多任务基准测试上的实验表明，与静态隔离和标准微调相比，EPI 持续减少了干扰和遗忘，同时提高了整体泛化能力。我们的分析强调了将隔离机制与学习多种能力时的演进动态同步的必要性。

## Abstract
Supervised Fine-Tuning (SFT) of large language models often suffers from task interference and catastrophic forgetting. Recent approaches alleviate this issue by isolating task-critical parameters during training. However, these methods represent a static solution to a dynamic problem, assuming that parameter importance remains fixed once identified. In this work, we empirically demonstrate that parameter importance exhibits temporal drift over the course of training. To address this, we propose Evolving Parameter Isolation (EPI), a fine-tuning framework that adapts isolation decisions based on online estimates of parameter importance. Instead of freezing a fixed subset of parameters, EPI periodically updates isolation masks using gradient-based signals, enabling the model to protect emerging task-critical parameters while releasing outdated ones to recover plasticity. Experiments on diverse multi-task benchmarks demonstrate that EPI consistently reduces interference and forgetting compared to static isolation and standard fine-tuning, while improving overall generalization. Our analysis highlights the necessity of synchronizing isolation mechanisms with the evolving dynamics of learning diverse abilities.

*   **研究背景**：有监督微调（SFT）是大语言模型（LLM）获取特定能力的关键。但在多任务或顺序学习场景下，模型常面临“跷跷板现象”（提升一项能力却损害另一项）和“灾难性遗忘”。
*   **核心问题**：现有的参数隔离技术（Parameter Isolation）通过锁定任务关键参数来减少干扰，但它们大多基于一个**静态假设**——即参数的重要性在初始化后保持不变。
*   **研究动机**：作者通过实证发现，参数重要性在微调过程中存在**时间漂移（Temporal Drift）**。静态掩码会锁定已过时的参数（浪费容量），且无法保护新出现的关键参数（导致遗忘）。

### 2. 方法论：演进式参数隔离（EPI）
EPI 框架的核心思想是同步隔离机制与模型的动态学习轨迹，主要包含以下三个关键技术环节：
*   **在线重要性估计（Fisher 近似）**：
    *   利用梯度平方的**指数移动平均（EMA）**来近似 Fisher 信息矩阵的对角元素。
    *   公式逻辑：$S_t = \beta S_{t-1} + (1 - \beta)(g_t \odot g_t)$。EMA 充当低通滤波器，过滤掉随机小批次采样带来的噪声，捕捉稳定的重要性趋势。
*   **自适应层级归一化（Layer-wise Normalization）**：
    *   由于深度网络中不同层的梯度量级差异巨大，全局排序会导致掩码偏向特定层。
    *   EPI 对每一层的参数得分进行 Min-Max 归一化，确保在统一的概率空间内进行全局 Top-p% 选择，实现跨层的公平筛选。
*   **动态掩码生成与演进**：
    *   设定更新周期 $H$（如每 500 步），定期刷新二值掩码 $m_t$。
    *   **演进逻辑**：新进入掩码的参数被锁定以保护知识，移出掩码的参数被释放以恢复可塑性，形成一个随训练轨迹移动的“动态盾牌”。

### 6. 主要结论与发现
*   **性能领先**：EPI 在所有基准模型上均显著优于标准 SFT 和静态隔离方法，尤其在推理密集型任务（如 GSM8K）上提升最明显。
*   **解决稳定性-可塑性困境**：EPI 证明了仅需锁定约 **1%** 的关键参数即可有效抑制遗忘，同时动态释放机制保证了模型学习新任务的灵活性。
*   **重要性确实在漂移**：实验显示，训练结束时的关键参数与初始时相比，重合度往往低于 50%，且高层参数的更新频率显著高于底层。

### 7. 优点与亮点
*   **动态视角**：挑战了领域内长期存在的静态参数重要性假设，更符合深度学习的非线性优化本质。
*   **无需额外参数**：与 LoRA 等 PEFT 方法不同，EPI 在原始参数空间操作，不改变模型结构，不增加推理开销。
*   **层级公平性**：引入层级归一化解决了深度网络中常见的梯度缩放偏差问题，这是一个简单但极其有效的改进。

### 8. 不足与局限
*   **内存开销**：虽然不增加计算量，但需要存储一份与模型参数等大的 EMA 统计量（类似于增加了一个 Adam 优化器状态），对于超大规模模型（如 30B+）可能会带来显存压力。
*   **超参数依赖**：隔离比例 $p$ 和更新间隔 $H$ 仍需手动设定。虽然实验证明了其鲁棒性，但在极端不同的任务分布下可能需要重新调优。
*   **语言覆盖**：实验主要集中在英语和结构化任务上，在多语言或极低资源场景下的表现尚待验证。
