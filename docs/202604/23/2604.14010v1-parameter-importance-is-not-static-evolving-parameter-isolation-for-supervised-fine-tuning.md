## 摘要
大语言模型的有监督微调（SFT）经常受到任务干扰和灾难性遗忘的困扰。最近的方法通过在训练期间隔离任务关键参数来缓解这一问题。然而，这些方法代表了针对动态问题的静态解决方案，假设参数重要性一旦确定就保持不变。在这项工作中，我们通过实证证明了参数重要性在训练过程中表现出时间漂移。为了解决这个问题，我们提出了演化参数隔离（EPI），这是一种根据参数重要性的在线估计来调整隔离决策的微调框架。EPI 不是冻结固定的参数子集，而是利用基于梯度的信号定期更新隔离掩码，使模型能够保护新出现的任务关键参数，同时释放过时的参数以恢复塑性。在多种多任务基准测试上的实验表明，与静态隔离和标准微调相比，EPI 一致地减少了干扰和遗忘，同时提高了整体泛化能力。我们的分析强调了将隔离机制与学习多种能力的演化动态同步的必要性。

## Abstract
Supervised Fine-Tuning (SFT) of large language models often suffers from task interference and catastrophic forgetting. Recent approaches alleviate this issue by isolating task-critical parameters during training. However, these methods represent a static solution to a dynamic problem, assuming that parameter importance remains fixed once identified. In this work, we empirically demonstrate that parameter importance exhibits temporal drift over the course of training. To address this, we propose Evolving Parameter Isolation (EPI), a fine-tuning framework that adapts isolation decisions based on online estimates of parameter importance. Instead of freezing a fixed subset of parameters, EPI periodically updates isolation masks using gradient-based signals, enabling the model to protect emerging task-critical parameters while releasing outdated ones to recover plasticity. Experiments on diverse multi-task benchmarks demonstrate that EPI consistently reduces interference and forgetting compared to static isolation and standard fine-tuning, while improving overall generalization. Our analysis highlights the necessity of synchronizing isolation mechanisms with the evolving dynamics of learning diverse abilities.

### 1. 论文的核心问题与整体含义
*   **研究背景**：有监督微调（SFT）是大语言模型（LLM）获取特定能力的关键。但在多任务或顺序学习场景下，模型常面临“跷跷板效应”（提升一项能力却损害另一项）和“灾难性遗忘”。
*   **核心问题**：现有的参数隔离技术（Parameter Isolation）通过锁定任务关键参数来缓解干扰，但它们大多基于一个**静态假设**——即参数的重要性在初始化后保持不变。
*   **核心发现**：论文通过实证研究发现，参数重要性在微调过程中存在**时间漂移（Temporal Drift）**。静态掩码会随着训练推进而逐渐失效：既保护了已过时的参数（浪费容量），又未能锁定新出现的关键参数（导致遗忘）。

### 2. 方法论：演化参数隔离（EPI）
EPI 框架旨在将隔离机制与模型的动态学习轨迹同步，核心步骤如下：
*   **在线重要性估计**：利用平方梯度的指数移动平均（EMA）来近似 Fisher 信息矩阵（FIM）的对角元素。公式为：$S_t = \beta S_{t-1} + (1 - \beta)(g_t \odot g_t)$。EMA 充当低通滤波器，过滤掉随机小批次采样带来的噪声，揭示稳定的重要性趋势。
*   **自适应层级归一化**：由于深度网络中不同层的梯度量级差异巨大，直接全局排序会导致保护机制偏向某些层。EPI 对每一层的参数进行 Min-Max 归一化，将重要性分数映射到统一的 $[0, 1]$ 空间，确保各层关键参数都能被公平筛选。
*   **动态掩码演化**：每隔 $H$ 个优化步，根据归一化后的分数重新选择 Top-$p\%$ 的参数生成二进制掩码 $m_t$。
*   **稀疏优化更新**：在 AdamW 更新阶段，仅对未被掩码锁定的参数进行更新：$\theta_{t+1} = \theta_t + (1 - m_t) \odot \Delta \theta_t$。这形成了一个“移动盾牌”，动态保护当前关键区域。

### 3. 实验设计
*   **数据集与场景**：涵盖数学推理（GSM8K）、逻辑推理（LogiQA）、代码生成（CodeAlpaca）和通用对话（Alpaca, UltraChat）。实验采用多阶段顺序微调设置，以模拟真实且具有挑战性的任务干扰场景。
*   **基座模型（Backbones）**：LLaMA-3-8B、Qwen2-7B、Gemma-2-9B 和 Mistral-7B。
*   **对比方法（Baselines）**：
    1.  **Full SFT**：全量多任务微调。
    2.  **Multi-Stage SFT**：随机分组或启发式分组的顺序微调。
    3.  **Static Isolation**：基于初始探测生成的固定掩码隔离方法。

### 4. 资源与算力
*   **硬件环境**：实验在 **8 台 NVIDIA A100 (80GB) GPU** 上进行。
*   **训练细节**：所有模型微调 3 个 Epoch，学习率为 $1 \times 10^{-5}$，Batch Size 为 64。静态隔离基准需要额外的 1 个 Epoch 进行探测运行。

### 5. 实验数量与充分性
*   **实验规模**：在 4 种不同架构的开源模型上进行了验证，涵盖了从推理到创作的 5 类异构任务。
*   **消融实验**：非常详尽。研究了层级归一化的必要性、EMA 平滑因子的影响、隔离比例 $p$ 的敏感性（发现 $1\%$ 是最佳平衡点）、以及掩码更新频率 $H$ 的影响。
*   **客观性**：所有实验均运行 3 次并报告均值与标准差，确保了结果的统计可靠性。通过 Jaccard 相似度定量分析了掩码的动态演化过程，证明了“重要性漂移”的真实存在。

### 6 主要结论与发现
*   **性能领先**：EPI 在所有基座模型上均显著优于标准 SFT 和静态隔离方法，尤其在推理密集型任务（如 GSM8K）上提升最明显。
*   **缓解遗忘**：遗忘率（FR）和任务梯度冲突（TGC）分析表明，EPI 能有效减少任务间的负面干扰。
*   **稳定性与塑性平衡**：EPI 证明了 LLM 的关键知识具有高度稀疏性（仅需保护 $1\%$ 的参数），且动态释放过时参数能显著提升模型学习新任务的“塑性”。

### 7. 优点
*   **动态视角**：挑战了参数重要性静态不变的传统认知，更符合深度学习的非线性优化本质。
*   **参数高效**：不需要引入任何额外的可训练参数（如 Adapter 或 LoRA 层），直接在原始参数空间操作。
*   **设计严谨**：层级归一化解决了深度网络中常见的梯度不平衡问题，EMA 机制保证了重要性估计的稳定性。

### 8. 不足与局限
*   **内存开销**：虽然不增加计算量，但维护 EMA 状态（$S_t$）需要与模型参数量同等的额外显存，对于超大规模模型（如 30B+）可能会带来显存压力。
*   **超参数依赖**：隔离比例 $p$ 和更新间隔 $H$ 仍需手动调优，虽然实验证明了其鲁棒性，但不同模型/任务的最优值可能存在差异。
*   **语言覆盖**：实验主要集中在英语语料和结构化推理任务，在多语言或极低资源场景下的表现尚待验证。
