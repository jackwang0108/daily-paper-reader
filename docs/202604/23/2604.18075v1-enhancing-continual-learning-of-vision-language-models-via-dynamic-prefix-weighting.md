## 摘要
我们研究了最近引入的视觉语言模型（VLMs）领域类增量学习场景。最近的研究通过使用参数高效的方法（如前缀微调或适配器）来应对这一挑战，这些方法通过加性向量将任务特定信息整合到输入标记中，从而促进模型对下游任务的适应。然而，以往的方法通常会对这些向量的权重进行归一化，忽略了不同输入标记需要不同程度调整的事实。为了解决这个问题，我们提出了动态前缀加权（DPW），这是一个动态分配前缀权重的框架，并辅以适配器。DPW 由以下部分组成：1）一个门控模块，根据相应输入标记的重要性调整每个前缀的权重；2）一种加权机制，将适配器输出权重推导为前缀微调权重的残差，确保仅在必要时使用适配器。实验结果表明，我们的方法在视觉语言模型的领域类增量学习场景中实现了最先进的性能。代码可在以下地址获取：https://github.com/YonseiML/dpw。

## Abstract
We investigate recently introduced domain-class incremental learning scenarios for vision-language models (VLMs). Recent works address this challenge using parameter-efficient methods, such as prefix-tuning or adapters, which facilitate model adaptation to downstream tasks by incorporating task-specific information into input tokens through additive vectors. However, previous approaches often normalize the weights of these vectors, disregarding the fact that different input tokens require different degrees of adjustment. To overcome this issue, we propose Dynamic Prefix Weighting (DPW), a framework that dynamically assigns weights to prefixes, complemented by adapters. DPW consists of 1) a gating module that adjusts the weights of each prefix based on the importance of the corresponding input token, and 2) a weighting mechanism that derives adapter output weights as a residual of prefix-tuning weights, ensuring that adapters are utilized only when necessary. Experimental results demonstrate that our method achieves state-of-the-art performance in domain-class incremental learning scenarios for VLMs. The code is available at: https://github.com/YonseiML/dpw.

论文聚焦于**视觉语言模型（VLM）的持续学习（Continual Learning, CL）**。
*   **核心挑战**：当预训练模型（如 CLIP）顺序学习新任务时，会面临“灾难性遗忘”以及“前向遗忘”（即零样本能力的下降）。
*   **研究动机**：现有的参数高效微调（PEFT）方法（如前缀微调 Prefix-tuning 或适配器 Adapters）通常在样本级别分配权重，对一个图像中的所有 Token（标记）施加统一的影响。作者指出，图像中不同 Token 对任务的贡献度不同，统一加权忽略了 Token 级别的细粒度需求。此外，预训练的注意力投影机制往往会模糊任务相关与无关 Token 的界限，导致知识迁移效率低下。

### 2. 论文提出的方法论：动态前缀加权（DPW）
作者提出了 **DPW 框架**，旨在实现 Token 级别的动态信息注入，其核心由三个模块组成：

*   **重参数化前缀注意力（RePA）**：
    *   **核心思想**：放弃传统的 Query-Key 点积计算，改用一个简单的仿射变换（Affine Transformation）直接从输入 Token 计算前缀得分。
    *   **技术细节**：通过将原始注意力公式中的投影矩阵和前缀向量合并为一个可学习的权重矩阵 $W_G$ 和偏置 $B_G$，使模型能更直接地捕捉 Token 的任务相关性，避免了预训练投影矩阵对 Token 特征的干扰。
*   **条件激活（CondAct）**：
    *   **核心思想**：替代传统的 Softmax 激活，以实现权重的动态范围控制。
    *   **技术细节**：包含**条件归一化**（将总权重限制在 [0, 1] 之间，防止过度拟合新任务而破坏零样本能力）和**条件过滤**（基于 [CLS] Token 的得分分布，利用高斯模型动态剔除不相关的前缀）。
*   **残差加权机制（RWM）**：
    *   **核心思想**：将适配器（LoRA）作为前缀微调的补充。
    *   **技术细节**：适配器的输出权重被定义为前缀权重的“残差”。当 Token 的前缀得分超过 1 时，多余的权重才分配给适配器。这种设计确保了适配器仅在必要时（即前缀无法充分表达任务信息时）介入，实现了前缀与适配器的协同。

### 6. 论文的主要结论与发现

*   **创新性**：首次在 VLM 持续学习中引入 Token 级别的动态加权，而非传统的样本级或任务级加权。

*   **鲁棒性**：通过 CondAct 机制，在提升新任务性能的同时，极好地保留了模型的零样本泛化能力。
