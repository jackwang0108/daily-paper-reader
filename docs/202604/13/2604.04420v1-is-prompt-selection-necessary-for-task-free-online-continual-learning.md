## 摘要
无任务在线持续学习最近已成为一种处理动态现实环境中持续学习问题的现实范式。在这些环境中，数据以非平稳流的形式到达，没有明确的任务边界，且只能被观察一次。为了应对这些具有挑战性的场景，许多近期的方法采用了提示选择（prompt selection），这是一种根据输入信号从提示池中选择提示的自适应策略。然而，我们观察到，此类选择策略往往无法选出合适的提示，尽管对关键参数进行了额外训练，但结果仍不尽如人意。受此启发，我们提出了一种简单而有效的 SinglePrompt，它消除了对提示选择的需求，并专注于分类器优化。具体而言，我们简单地：(i) 在每个自注意力块中注入单个提示；(ii) 采用基于余弦相似度的 Logit 设计，以缓解分类器权重中固有的遗忘效应；(iii) 对当前微批次（minibatch）中未出现的类别屏蔽其 Logit。凭借这种简单的无任务设计，我们的框架在各种在线持续学习基准测试中均达到了最先进的性能。源代码可在 https://github.com/efficient-learning-lab/SinglePrompt 获取。

## Abstract
Task-free online continual learning has recently emerged as a realistic paradigm for addressing continual learning in dynamic, real-world environments, where data arrive in a non-stationary stream without clear task boundaries and can only be observed once. To consider such challenging scenarios, many recent approaches have employed prompt selection, an adaptive strategy that selects prompts from a pool based on input signals. However, we observe that such selection strategies often fail to select appropriate prompts, yielding suboptimal results despite additional training of key parameters. Motivated by this observation, we propose a simple yet effective SinglePrompt that eliminates the need for prompt selection and focuses on classifier optimization. Specifically, we simply (i) inject a single prompt into each self-attention block, (ii) employ a cosine similarity-based logit design to alleviate the forgetting effect inherent in the classifier weights, and (iii) mask logits for unexposed classes in the current minibatch. With this simple task-free design, our framework achieves state-of-the-art performance across various online continual learning benchmarks. Source code is available at https://github.com/efficient-learning-lab/SinglePrompt.

### 2. 论文提出的方法论：SinglePrompt
作者提出了 **SinglePrompt** 框架，其核心思想是“化繁为简”，将重点从“选择提示”转向“优化分类器”。
*   **核心思想**：取消提示池和选择机制，在预训练 Vision Transformer (ViT) 的前 $K$ 层中为每个自注意力块注入单一且固定的可学习提示。
*   **关键技术细节**：
    1.  **在线适配的前缀微调 (Prefix Tuning)**：在自注意力操作中，将可学习的提示向量 $p_k$ 和 $p_v$ 分别拼接到 Key 和 Value 序列前，仅更新这些提示参数，冻结 ViT 主干。
    2.  **基于余弦相似度的 Logit 计算**：传统线性分类器权重的 $L_2$ 范数在持续学习中会失衡（早期类别范数变小导致遗忘）。SinglePrompt 改用余弦相似度计算 Logit，消除权重范数的影响，公式为：$z_c = \frac{g \cdot c_c}{\|g\|\|c_c\|} \cdot \frac{1}{\tau}$。
    3.  **微批次逻辑值掩码 (Minibatch Logit Masking)**：在训练时，仅对当前微批次中出现的类别计算交叉熵损失，屏蔽（Mask）未出现类别的 Logit，防止未见类别的原型权重被错误更新。

### 6. 论文的主要结论与发现
*   **提示选择并非必要**：在无任务在线持续学习中，复杂的提示选择机制不仅增加了参数量，还可能因为选择错误引入噪声。
*   **分类器优化更关键**：通过余弦相似度分类器和 Logit 掩码，可以比复杂的提示池机制更有效地缓解“灾难性遗忘”。

*   **简洁高效**：摒弃了复杂的查询-键（Query-Key）匹配机制，架构极简，易于实现。
*   **参数高效**：相比于维护庞大的提示池，单一提示显著降低了可学习参数量。
*   **理论洞察**：深入揭示了现有提示选择方法的失效现象，为持续学习研究提供了新的视角。

*   **架构依赖性**：该方法高度依赖于预训练的 Vision Transformer (ViT)，对于非 Transformer 架构（如 ResNet）的适用性讨论较少。
