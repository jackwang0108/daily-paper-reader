## 摘要
大语言模型的有监督微调（SFT）通常面临任务干扰和灾难性遗忘的问题。近期的研究方法通过在训练过程中隔离任务关键参数来缓解这一问题。然而，这些方法是针对动态问题的静态解决方案，假设参数重要性一旦确定便保持不变。在本研究中，我们通过实证研究证明，参数重要性在训练过程中表现出时间偏移。为解决这一问题，我们提出了演进式参数隔离（Evolving Parameter Isolation, EPI），这是一种根据参数重要性的在线评估来调整隔离决策的微调框架。EPI 并非冻结固定的参数子集，而是利用基于梯度的信号定期更新隔离掩码，使模型能够保护新出现的任务关键参数，同时释放过时的参数以恢复可塑性。在多种多任务基准测试上的实验表明，与静态隔离和标准微调相比，EPI 持续减少了干扰和遗忘，同时提高了整体泛化能力。我们的分析强调了将隔离机制与学习多种能力时的演进动态同步的必要性。

## Abstract
Supervised Fine-Tuning (SFT) of large language models often suffers from task interference and catastrophic forgetting. Recent approaches alleviate this issue by isolating task-critical parameters during training. However, these methods represent a static solution to a dynamic problem, assuming that parameter importance remains fixed once identified. In this work, we empirically demonstrate that parameter importance exhibits temporal drift over the course of training. To address this, we propose Evolving Parameter Isolation (EPI), a fine-tuning framework that adapts isolation decisions based on online estimates of parameter importance. Instead of freezing a fixed subset of parameters, EPI periodically updates isolation masks using gradient-based signals, enabling the model to protect emerging task-critical parameters while releasing outdated ones to recover plasticity. Experiments on diverse multi-task benchmarks demonstrate that EPI consistently reduces interference and forgetting compared to static isolation and standard fine-tuning, while improving overall generalization. Our analysis highlights the necessity of synchronizing isolation mechanisms with the evolving dynamics of learning diverse abilities.

### 1. 核心问题与研究动机
*   **核心问题**：在大语言模型（LLM）的有监督微调（SFT）中，如何解决多任务学习带来的**任务干扰（Task Interference）**和**灾难性遗忘（Catastrophic Forgetting）**。
*   **研究背景**：现有的“参数隔离”技术通常假设参数的重要性在微调开始后是固定不变的（静态隔离）。
*   **核心发现**：作者通过实证研究发现，参数重要性在训练过程中会发生**时间偏移（Parameter Importance Drift）**。例如，早期负责格式学习的参数在后期可能变得不再重要，而负责复杂推理的参数则在后期才凸显其关键性。静态隔离会导致模型保护了过时的参数，却未能锁定新出现的关键参数。

### 2. 方法论：演进式参数隔离（EPI）
EPI 框架的核心思想是同步隔离机制与模型的动态学习轨迹，其关键技术细节包括：
*   **在线重要性估计**：利用 Fisher 信息矩阵的对角近似，通过计算平方梯度的**指数移动平均（EMA）**来实时追踪参数敏感度。EMA 充当低通滤波器，过滤掉随机小批量采样带来的噪声。
*   **自适应层级归一化（Layer-wise Normalization）**：由于深度网络中不同层的梯度量级差异巨大，直接全局排序会导致保护机制偏向某些层。EPI 对每一层的得分进行 Min-Max 归一化，确保参数选择在各层之间是公平且无偏的。
*   **动态掩码演进**：设定更新间隔 $H$（如每 500 步），定期根据归一化后的得分重新生成二进制掩码。该机制实现了“移动盾牌”的效果：锁定当前关键参数（保护稳定性），释放冗余参数（恢复可塑性）。
*   **优化集成**：在 AdamW 优化器更新阶段应用掩码，被保护的参数不接受梯度更新，也不进行权重衰减。

### 3. 实验设计
*   **数据集与场景**：涵盖了数学推理（GSM8K）、逻辑推理（LogiQA）、代码生成（CodeAlpaca）以及通用指令遵循（Alpaca, UltraChat）。实验采用多阶段顺序微调，以模拟最严苛的遗忘场景。
*   **基座模型（Backbones）**：选择了四个主流开源模型：LLaMA-3-8B、Qwen2-7B、Gemma-2-9B 和 Mistral-7B。
*   **对比方法（Baselines）**：
    1.  **Full SFT**：全量参数联合微调。
    2.  **Multi-Stage SFT**：分为随机分组和启发式分组的顺序微调。
    3.  **Static Isolation**：基于初始探测生成的固定掩码隔离方法。

*   **硬件环境**：实验在 **8 张 NVIDIA A100 (80GB) GPU** 上进行。

### 5. 实验数量与充分性
*   **实验规模**：论文进行了大规模的跨模型验证（4 种架构），并针对每个任务报告了标准化得分。
*   **消融实验**：非常详尽，涵盖了层级归一化的必要性、EMA 平滑因子的影响、隔离比例 $p$ 的敏感性、掩码更新频率 $H$ 的影响。
*   **分析维度**：除了性能指标，还深入分析了遗忘率（FR）、梯度冲突（TGC）、掩码的 Jaccard 相似度（量化漂移）以及层级翻转率。

### 6. 主要结论与发现
*   **性能领先**：EPI 在所有基座模型上均刷新了 SFT 的表现，尤其在推理密集型任务（如 GSM8K）上提升显著（比静态隔离高出约 2.6%-2.8%）。
*   **缓解遗忘**：EPI 显著降低了遗忘率和任务间的梯度冲突（跷跷板效应）。
*   **参数稀疏性**：实验发现最佳隔离比例 $p$ 仅为 **1%** 左右，证明 LLM 的任务关键知识高度稀疏。
*   **动态性验证**：训练结束时，掩码与初始状态的重合度往往低于 50%，证实了“参数重要性并非静态”的假设。

### 7. 优点与亮点
*   **视角新颖**：挑战了参数隔离领域长期存在的“静态重要性”假设，提出了动态演进的视角。
*   **高效简洁**：无需引入额外的可训练参数（如 Adapter 或 LoRA），也不需要复杂的任务路由机制。
*   **即插即用**：作为一种优化器层面的改进，可以轻易集成到现有的微调管线中。
*   **理论支撑**：结合了 Fisher 信息矩阵和损失景观几何学（Hessian 矩阵）的理论推导。

### 8. 不足与局限
*   **内存开销**：虽然计算量增加极小，但需要存储与模型参数量同规模的 EMA 状态（St），对于显存极度紧张的场景（如 30B+ 模型）会有一定压力。
*   **超参数依赖**：隔离比例 $p$ 和更新间隔 $H$ 仍需手动设定，虽然实验证明其具有一定的鲁棒性，但不同规模模型可能需要微调。
*   **任务覆盖**：实验主要集中在英文和结构化推理任务，在多语言、低资源语言或多模态任务上的表现尚待验证。
*   **任务相似度**：目前尚未显式建模任务间的层级关系或相似度，仅依赖梯度信号进行启发式隔离。
