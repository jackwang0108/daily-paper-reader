## 摘要
大语言模型的监督微调（SFT）通常面临任务干扰和灾难性遗忘的问题。最近的方法通过在训练期间隔离任务关键参数来缓解这一问题。然而，这些方法代表了针对动态问题的静态解决方案，假设参数重要性一旦确定就保持不变。在这项工作中，我们通过实证证明了参数重要性在训练过程中表现出时间漂移。为了解决这个问题，我们提出了演化参数隔离（EPI），这是一种根据参数重要性的在线估计来调整隔离决策的微调框架。EPI 不是冻结固定的参数子集，而是利用基于梯度的信号定期更新隔离掩码，使模型能够保护新出现的任务关键参数，同时释放过时的参数以恢复可塑性。在多种多任务基准测试上的实验表明，与静态隔离和标准微调相比，EPI 持续减少了干扰和遗忘，同时提高了整体泛化能力。我们的分析强调了将隔离机制与学习多种能力的演化动态同步的必要性。

## Abstract
Supervised Fine-Tuning (SFT) of large language models often suffers from task interference and catastrophic forgetting. Recent approaches alleviate this issue by isolating task-critical parameters during training. However, these methods represent a static solution to a dynamic problem, assuming that parameter importance remains fixed once identified. In this work, we empirically demonstrate that parameter importance exhibits temporal drift over the course of training. To address this, we propose Evolving Parameter Isolation (EPI), a fine-tuning framework that adapts isolation decisions based on online estimates of parameter importance. Instead of freezing a fixed subset of parameters, EPI periodically updates isolation masks using gradient-based signals, enabling the model to protect emerging task-critical parameters while releasing outdated ones to recover plasticity. Experiments on diverse multi-task benchmarks demonstrate that EPI consistently reduces interference and forgetting compared to static isolation and standard fine-tuning, while improving overall generalization. Our analysis highlights the necessity of synchronizing isolation mechanisms with the evolving dynamics of learning diverse abilities.

*   **核心目标**：打破静态隔离的局限，提出一种能够随训练进程动态调整保护对象的微调框架，以平衡模型的“稳定性”（保留旧知识）与“塑性”（吸收新能力）。

### 2. 方法论：演化参数隔离（EPI）
EPI 框架的核心思想是同步隔离机制与模型的实时学习轨迹。其主要技术流程如下：
*   **在线重要性估计**：利用梯度信号的平方来近似 Fisher 信息矩阵（FIM）的对角元素，从而衡量参数对损失函数的敏感度。为了过滤随机小批量采样带来的噪声，引入了**指数移动平均（EMA）**来平滑梯度信号。
*   **自适应层级归一化（Layer-wise Normalization）**：由于深度网络中不同层的梯度量级存在显著差异（梯度缩放不匹配），直接进行全局排名会导致保护机制偏向某些特定层。EPI 对每一层的参数重要性得分进行 Min-Max 归一化，确保各层都能公平地选出关键参数。
*   **动态掩码演化**：系统每隔 $H$ 个优化步更新一次二进制掩码。通过选取归一化得分最高的 Top-$p\%$ 参数进行锁定（禁止更新），同时释放那些重要性下降的参数。
*   **稀疏优化集成**：在 AdamW 优化器更新阶段，将演化掩码应用于最终的参数增量，确保受保护的参数不被修改，且不受到权重衰减的影响。

### 6. 主要结论与发现
*   **性能领先**：EPI 在所有基座模型和任务上均显著优于标准 SFT 和静态隔离方法，尤其在推理密集型任务（如 GSM8K）上提升最明显。

*   **层级差异**：高层（靠近输出层）的参数重要性漂移比底层更剧烈，说明高层在适应新任务时需要更高的塑性。

### 7. 优点与亮点
*   **动态视角**：首次系统性地识别并量化了 SFT 过程中的“参数重要性漂移”，挑战了领域内的静态建模惯例。
*   **即插即用**：EPI 不需要引入额外的可训练参数（如 Adapter 或 LoRA 模块），直接在原始参数空间操作，保持了模型的原生结构。
*   **设计精巧**：通过层级归一化解决了深度网络中普遍存在的梯度偏差问题，使参数保护更加精准。

### 8. 不足与局限
*   **内存开销**：虽然计算量增加不多，但需要额外存储一份与模型参数量一致的 EMA 状态，对于显存受限的场景（如微调 30B 以上模型）可能存在压力。

*   **覆盖范围**：实验主要集中在英语任务和结构化推理上，对于多语言场景、多模态数据或极长序列任务的有效性尚待验证。
