## 摘要
基于适配器（Adapter）的方法已成为大语言模型（LLM）持续学习（CL）的一种具有成本效益的方法，它通过为每个任务顺序学习一个低秩更新矩阵来实现。为了缓解灾难性遗忘，最先进的方法通过针对子空间或坐标维度的干扰，对新适配器相对于旧适配器施加约束。在本文中，我们提出了 JumpLoRA，这是一个通过使用 JumpReLU 门控在低秩自适应（LoRA）模块中自适应诱导稀疏性的新框架。该方法实现了动态参数隔离，有助于防止任务干扰。我们证明了我们的方法具有高度模块化，并与基于 LoRA 的持续学习方法兼容。具体而言，它显著提升了 IncLoRA 的性能，并超越了领先的先进持续学习方法 ELLA。

## Abstract
Adapter-based methods have become a cost-effective approach to continual learning (CL) for Large Language Models (LLMs), by sequentially learning a low-rank update matrix for each task. To mitigate catastrophic forgetting, state-of-the-art approaches impose constraints on new adapters with respect to the previous ones, by targeting either subspace or coordinate-wise interference. In this paper, we propose JumpLoRA, a novel framework to adaptively induce sparsity in the Low-Rank Adaptation (LoRA) blocks through the use of JumpReLU gating. The method achieves dynamic parameter isolation, which helps prevent task interference. We demonstrate that our method is highly modular and compatible with LoRA-based CL approaches. Specifically, it significantly boosts the performance of IncLoRA and outperforms the leading state-of-the-art CL method, ELLA.

### 1. 核心问题与整体含义（研究动机和背景）
大语言模型（LLMs）在顺序学习新任务时，面临严重的**灾难性遗忘**问题，即学习新知识会导致旧知识的丧失。虽然基于 LoRA（低秩自适应）的参数高效微调（PEFT）方法降低了计算成本，但简单的 LoRA 堆叠在持续学习（CL）中仍会产生**任务间干扰**。
*   **研究动机**：现有的方法（如 ELLA 或 InfLoRA）通过正交投影或坐标约束来减少干扰，但它们通常在稠密参数空间操作。随着任务增加，这种稠密更新会面临容量饱和及残余梯度干扰。
*   **核心目标**：通过引入**自适应稀疏性**，实现参数级别的动态隔离，让不同任务的适配器占用互不重叠的参数坐标，从而彻底减少干扰。

### 2. 方法论：核心思想与关键技术
论文提出了 **JumpLoRA** 框架，其核心是将 **JumpReLU** 门控机制应用于 LoRA 的权重更新。

*   **核心思想**：在训练 LoRA 适配器（$\Delta W = A \cdot B$）时，不仅学习低秩矩阵 $A$ 和 $B$，还学习一个层内或块内的**阈值 $\tau$**。只有当权重更新的幅度超过 $\tau$ 时，该更新才会被保留，否则被置为零。
*   **关键技术细节**：
    1.  **JumpReLU 门控**：定义为 $JumpReLU_\tau(x) = x \cdot H(x - \tau)$，其中 $H$ 是阶跃函数。
    2.  **直通估计器（STE）**：由于阶跃函数不可导，使用伪导数（Straight-Through Estimator）来允许通过反向传播优化阈值 $\tau$。
    3.  **平滑调度机制（Scheduling）**：为了避免初始化时（$\Delta W=0$）梯度被完全截断，引入了插值系数 $\gamma$。训练初期进行全量更新，随着训练进行，逐渐过渡到完全稀疏的 JumpReLU 更新。
    4.  **对称处理**：为了保留正负向的大幅更新，实际应用为对 $|\Delta W|$ 进行阈值处理。
*   **算法流程**：注入 LoRA -> 预热训练（无稀疏） -> 初始化 $\tau$ -> 线性增加稀疏比例 -> 最终获得稀疏适配器并合并至基座模型。
