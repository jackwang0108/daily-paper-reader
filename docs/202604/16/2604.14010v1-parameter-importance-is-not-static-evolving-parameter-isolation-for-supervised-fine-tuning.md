## 摘要
大语言模型的有监督微调（SFT）通常面临任务干扰和灾难性遗忘的问题。近期的研究方法通过在训练过程中隔离任务关键参数来缓解这一问题。然而，这些方法是针对动态问题的静态解决方案，假设参数重要性一旦确定便保持不变。在本研究中，我们通过实证证明了参数重要性在训练过程中表现出时间偏移。为此，我们提出了演进式参数隔离（Evolving Parameter Isolation, EPI），这是一种根据参数重要性的在线评估来调整隔离决策的微调框架。EPI 并非冻结固定的参数子集，而是利用基于梯度的信号定期更新隔离掩码，使模型能够保护新出现的任务关键参数，同时释放过时的参数以恢复塑性。在多种多任务基准测试上的实验表明，与静态隔离和标准微调相比，EPI 持续减少了干扰和遗忘，同时提高了整体泛化能力。我们的分析强调了将隔离机制与学习多种能力的演进动态同步的必要性。

## Abstract
Supervised Fine-Tuning (SFT) of large language models often suffers from task interference and catastrophic forgetting. Recent approaches alleviate this issue by isolating task-critical parameters during training. However, these methods represent a static solution to a dynamic problem, assuming that parameter importance remains fixed once identified. In this work, we empirically demonstrate that parameter importance exhibits temporal drift over the course of training. To address this, we propose Evolving Parameter Isolation (EPI), a fine-tuning framework that adapts isolation decisions based on online estimates of parameter importance. Instead of freezing a fixed subset of parameters, EPI periodically updates isolation masks using gradient-based signals, enabling the model to protect emerging task-critical parameters while releasing outdated ones to recover plasticity. Experiments on diverse multi-task benchmarks demonstrate that EPI consistently reduces interference and forgetting compared to static isolation and standard fine-tuning, while improving overall generalization. Our analysis highlights the necessity of synchronizing isolation mechanisms with the evolving dynamics of learning diverse abilities.

论文聚焦于大语言模型（LLM）在**有监督微调（SFT）**阶段面临的两个核心挑战：**任务干扰（Task Interference）**和**灾难性遗忘（Catastrophic Forgetting）**。
*   **研究动机**：现有的“参数隔离”技术（如 CPI-FT）通过锁定任务关键参数来保护知识，但它们基于一个未经检验的假设——参数重要性在训练开始后是固定不变的。
*   **核心发现**：作者通过实证研究发现，参数重要性在微调过程中存在**时间偏移（Temporal Drift）**。早期关键的参数（如负责格式的）后期可能变得冗余，而复杂的推理逻辑参数可能在训练后期才变得关键。因此，静态的隔离掩码会逐渐与学习轨迹脱节，导致“过度保护”旧参数或“未能保护”新出现的关键参数。

### 2. 方法论：演进式参数隔离（EPI）
EPI 框架的核心思想是**将隔离机制与模型的动态优化轨迹同步**，其关键技术流程如下：
*   **在线重要性估计**：利用梯度信号的平方作为 Fisher 信息矩阵（FIM）的对角线近似，衡量参数敏感度。为了过滤随机小批次采样带来的噪声，引入了**指数移动平均（EMA）**来平滑梯度信号，捕捉稳定的重要性趋势。
*   **自适应层级归一化（Layer-wise Normalization）**：由于深度网络中不同层的梯度量级差异巨大，直接进行全局排名会导致保护机制偏向某些特定层。EPI 对每一层的参数重要性得分进行 Min-Max 归一化，确保各层参数在统一的 [0, 1] 概率空间内竞争。
*   **动态掩码演进**：系统设定一个更新周期 $H$（如每 500 步）。在更新点，根据归一化得分选取 Top-$p$% 的参数生成新的二进制掩码 $m_t$。
*   **稀疏优化更新**：在 AdamW 优化器更新时，应用掩码 $m_t$。被掩码覆盖的参数（$m_t=1$）将被锁定（不更新且不应用权重衰减），其余参数正常更新。这形成了一个“移动的盾牌”，动态保护当前最关键的知识。

### 3. 实验设计
*   **数据集与场景**：涵盖了数学推理（GSM8K）、逻辑推理（LogiQA）、代码生成（CodeAlpaca）以及开放式指令遵循（Alpaca, UltraChat）。实验采用多阶段顺序微调，以模拟真实世界中异构任务流的挑战。
*   **基座模型（Backbones）**：选用了 LLaMA-3-8B、Qwen2-7B、Gemma-2-9B 和 Mistral-7B 四种主流开源模型。

    2.  **Multi-Stage SFT**：分为随机分组和启发式分组的顺序微调。

*   **实验规模**：论文在 4 个不同架构的基座模型上进行了主实验，并在 LLaMA-3 和 Gemma-2 上进行了详尽的消融研究。

### 6. 主要结论与发现
*   **性能领先**：EPI 在所有基座模型和任务上均显著优于 Full SFT 和静态隔离方法，尤其在推理密集型任务（如 GSM8K）上提升最明显。
*   **解决稳定性-塑性困境**：EPI 能够有效缓解“跷跷板效应”，在保护旧知识（低遗忘率）的同时，释放冗余空间以学习新能力（高塑性）。
*   **参数稀疏性**：实验发现 $p=1\%$ 是最佳隔离比例，证明 LLM 的任务关键知识具有极高的稀疏性。
*   **掩码动态性验证**：分析显示，训练结束时的掩码与初始掩码的重合度往往低于 50%，且高层参数的更新频率（Flip Rate）显著高于底层。

### 7. 优点
*   **动态视角**：首次系统性地识别并量化了 SFT 过程中的“参数重要性偏移”现象，挑战了传统静态隔离的认知。
*   **无需额外参数**：与 LoRA 或 Adapter 等 PEFT 方法不同，EPI 在原始参数空间操作，不增加推理延迟。
*   **设计精巧**：通过层级归一化解决了深度网络固有的梯度不平衡问题，使保护机制更加公平有效。

### 8. 不足与局限
*   **内存开销**：虽然不增加训练参数，但需要存储一份与模型参数等大的敏感度累加向量 $S_t$（类似 Adam 的动量状态），对于超大规模模型（如 30B+）会增加显存压力。
*   **超参数依赖**：隔离比例 $p$ 和更新间隔 $H$ 仍需手动调节，缺乏一种根据损失函数地形自动调整这些超参数的自适应机制。
*   **覆盖范围**：实验主要集中在英语任务和结构化推理，尚未在多语言场景或极长文本序列任务中验证其泛化性。
