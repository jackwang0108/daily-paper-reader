## 摘要
无任务在线持续学习（Task-free online continual learning）最近已成为解决动态现实环境中持续学习的一种现实范式。在这些环境中，数据以非平稳流的形式到达，没有明确的任务边界，且只能被观察一次。为了应对这些具有挑战性的场景，许多近期的方法采用了提示选择（prompt selection），这是一种根据输入信号从提示池中选择提示的自适应策略。然而，我们观察到，此类选择策略往往无法选择合适的提示，尽管对关键参数进行了额外训练，但结果仍不尽如人意。受此启发，我们提出了一种简单而有效的 SinglePrompt，它消除了对提示选择的需求，并专注于分类器优化。具体而言，我们简单地：(i) 在每个自注意力块中注入单个提示；(ii) 采用基于余弦相似度的 logit 设计，以减轻分类器权重中固有的遗忘效应；(iii) 对当前小批量（minibatch）中未出现的类别屏蔽其 logit。凭借这种简单的无任务设计，我们的框架在各种在线持续学习基准测试中均达到了最先进的性能。源代码可在 https://github.com/efficient-learning-lab/SinglePrompt 获取。

## Abstract
Task-free online continual learning has recently emerged as a realistic paradigm for addressing continual learning in dynamic, real-world environments, where data arrive in a non-stationary stream without clear task boundaries and can only be observed once. To consider such challenging scenarios, many recent approaches have employed prompt selection, an adaptive strategy that selects prompts from a pool based on input signals. However, we observe that such selection strategies often fail to select appropriate prompts, yielding suboptimal results despite additional training of key parameters. Motivated by this observation, we propose a simple yet effective SinglePrompt that eliminates the need for prompt selection and focuses on classifier optimization. Specifically, we simply (i) inject a single prompt into each self-attention block, (ii) employ a cosine similarity-based logit design to alleviate the forgetting effect inherent in the classifier weights, and (iii) mask logits for unexposed classes in the current minibatch. With this simple task-free design, our framework achieves state-of-the-art performance across various online continual learning benchmarks. Source code is available at https://github.com/efficient-learning-lab/SinglePrompt.

## 1. 核心问题与研究动机
*   **研究背景**：在线持续学习（Online Continual Learning, OCL）要求模型在非平稳的数据流中学习，且数据只能观察一次。为了应对任务边界模糊的挑战，近期主流方法（如 L2P, MISA）多采用“提示选择（Prompt Selection）”策略，即维护一个提示池并根据输入动态选择提示词。
*   **核心问题**：作者通过深入分析发现，现有的提示选择机制在无任务场景下表现极差——选择结果与输入类别几乎不相关，且存在提示词利用率低、任务识别准确率不足 50% 等问题。
*   **研究动机**：基于“提示选择往往失效”的观察，作者质疑复杂提示池的必要性，试图探索一种更简单、更鲁棒的无任务 OCL 框架。

## 2. 论文提出的方法论：SinglePrompt
SinglePrompt 摒弃了复杂的提示池和选择逻辑，核心思想是“回归单一提示并优化分类器”。
*   **Prefix Tuning（前缀微调）**：在预训练 Vision Transformer (ViT) 的前 $K$ 个自注意力块中，直接注入一对固定的、可学习的键（Key）和值（Value）提示词，不再进行动态选择。
*   **余弦相似度分类器**：传统线性分类器在持续学习中，早期类别的权重范数会变小导致遗忘。SinglePrompt 改用特征向量与类原型之间的余弦相似度来计算 Logits，消除权重范数不均的影响。
*   **小批量 Logit 掩码（Minibatch Logit Masking）**：在训练阶段，仅对当前小批量（Minibatch）中出现的类别计算交叉熵损失，屏蔽未出现类别的 Logits，防止无关类原型的错误更新。

## 6. 论文的主要结论与发现
*   **提示选择并非必要**：在无任务 OCL 中，复杂的提示选择机制不仅难以训练，且对性能贡献微乎其微。
*   **简化带来提升**：SinglePrompt 在参数量减少约 **60%-75%** 的情况下，性能全面超越了现有的 SOTA 方法。
*   **分类器设计至关重要**：余弦相似度分类器和小批量掩码是缓解在线学习中“灾难性遗忘”的关键，其重要性超过了提示词的复杂程度。

## 7. 优点
*   **简洁性**：提出了一种极简的框架，证明了在 OCL 领域“少即是多（Less is More）”。
*   **高性能**：在 ImageNet-R 等复杂数据集上，比之前的 SOTA 方法（MISA）在最终准确率上提升了约 **15%**。
*   **高效率**：大幅降低了可学习参数量和训练/推理时间，更符合“在线”学习的实时性要求。

## 8. 不足与局限
*   **预训练依赖**：该方法依赖于在 ImageNet 上预训练好的 ViT 骨干网络，若在缺乏高质量预训练模型的领域（如某些医学影像或特殊工业数据），其效果可能受限。

*   **任务类型单一**：实验主要集中在图像分类任务，尚未验证其在目标检测、分割或多模态持续学习任务中的泛化能力。
